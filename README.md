react 学习项目
第一周: 基础语法
    jsx 语法 
    函数组件/类组件 : 组件的两种实现方式
    state : 组件内部的状态数据
    props : 组件外部传递给组件的属性
    生命周期钩子 : 组件挂载、更新、卸载等过程中执行的函数
第二周: 组件通信
    父传子 : props
    子传父 : 回调函数
    状态提升 : 子组件通过状态提升将数据传递给共同的父组件
    context 机制 : 跨组件通信
    useEffect以及useEffect清理副作用 : 组件挂载完毕后执行
    自定义hooks
    Redux : 状态管理
    路由 : 页面跳转
    useMemo : 缓存计算结果,保持引用稳定
    useReducer : 较为复杂的状态数据管理
    useCallback : 缓存函数
    React.memo :　允许组件props不变时不发生渲染
